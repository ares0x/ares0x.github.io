# 


[gotests](https://github.com/cweill/gotests), 一个自动生成测试用例的工具
```shell
# 安装
go get -u github.com/cweill/gotests/...

# 使用
# 进入测试代码所在目录
gotests -all -w .
```

[gomock](https://github.com/golang/mock)，一个 mock 框架
```
$ go get github.com/golang/mock/gomock
$ go install github.com/golang/mock/mockgen
```
